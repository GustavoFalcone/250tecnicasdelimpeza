# Landing Page - +250 Técnicas de Limpeza Profissional

Landing page criada em projeto separado do entregável.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Checkouts

Os links ficam centralizados em:

`src/config/offer.js`

Substitua:

- `COLE_AQUI_O_LINK_DO_CHECKOUT_BASICO`
- `COLE_AQUI_O_LINK_DO_CHECKOUT_COMPLETO`
- `COLE_AQUI_O_LINK_DO_CHECKOUT_BASICO_DESCONTO`
- `COLE_AQUI_O_LINK_DO_CHECKOUT_COMPLETO_DESCONTO`
- `COLE_AQUI_O_LINK_DO_CHECKOUT_COMPLETO_POPUP`

## Imagens

As imagens também ficam em `src/config/offer.js` e estão configuradas com URLs diretas do Postimg.

## Git

Este projeto não mantém o `.git` do repositório usado como base. Configure um novo repositório antes de fazer qualquer push.
